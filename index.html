Git starter commands

starter commands

git init Initialise un repository local

git config --local user.name <user name> Initialise un email d'utilisateur

Add

git add <filename> Ajoute un fichier dans l'index

git add . Ajoute les fichiers du dossier en cours dans l'index

git add -A Ajoute tous les fichiers dans l'index

Commit

git commit -m "your message" Commit des fichiers de la zone staging vers le repository

git commit -a Commit es fichiers de la zone staging vers le repository et ouvre l'editeur par defaut pour editer le message

git add -A Ajoute tous les fichiers dans l'index

Redact Commit message

  • Prendre le temps de rédiger son message
  • ajouter un sujet
  • limiter le message à 50 caractères
  • Ecrire sans accents
  • Répondre à la question "Pourquoi ?" plutôt que "Comment ?"

Example Commit message

  • Calque :
  • ajouter nu nouveau calque à mon fichier index.html
